Файлы user_calculator.py и modern_calculator.py содержат работающий код для создания калькулятора. 
Простейший калькулятор user_calculator.py, к нему в файле test_calculator.py есть автотесты.
Файл calculator.py вспомогательный, к нему идет обращение в файле test_calculator.py.
Калькулятор из файла modern_calculator.py открывается в новом окне как полноценная программа, есть возможность вводить цифры, знаки, стирать запись с помощью кнопок.
